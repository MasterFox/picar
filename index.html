PiCar

Welcome to the project page of PiCar! PiCar allows you to control and program your radio-controlled car with a RasPi and some Python. Just fork the repository and try it out! Also follow @picar on Twitter!

Features of PiCar 0.01alpha (release on October 19, 2014)

  • Added MoviX engine
  • Added basic motor control
  • Added lighting control
